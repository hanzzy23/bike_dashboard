# bike_dashboard
## Persiapan Environment - Python
1. Install Python 3.10+
2. Cek instalasi di terminal/PowerShell
(python --version)
(pip --version)
3. Buat folder proyek Misal: C:\bike_dashboard lalu masukkan file day.csv dan hour.csv
4. Install library yang Dibutuhkan (pip install pandas matplotlib seaborn streamlit)

## Analisis Data
1. Import library dan Load Data
2. Lihat info dan head dataset
3. Cek Missing values dan lain nya

## Visualization dan Explanatory Analysis
- Pertanyaan 1: Hari atau jam penyewaan tertinggi
- Pertanyaan 2: Perbedaan weekday vs weekend
